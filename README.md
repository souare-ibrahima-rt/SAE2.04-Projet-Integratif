# Projet intégratif — SAE2.04

**Ibrahima SOUARE** · BUT R&T · IUT de Colmar · 2023-2024

## Le projet

Projet intégratif de fin de 1ère année, dans le cadre fictif d'un atelier coopératif d'artistes ayant besoin d'une infrastructure informatique complète.

## Partie infrastructure

Déploiement d'un contrôleur de domaine Active Directory avec Samba4 dans un conteneur Docker : création d'un réseau isolé, installation des paquets nécessaires, provisionnement du domaine, création des groupes et comptes utilisateurs en ligne de commande.

## Partie création artistique

Reproduction puis variation d'une œuvre générative inspirée de François Morellet, d'abord en Python avec matplotlib (grille de motifs géométriques aléatoires), puis portée en JavaScript avec Canvas pour une version web interactive.

## Stack technique

Samba4 · Docker · Python · matplotlib · JavaScript · Canvas · Active Directory
